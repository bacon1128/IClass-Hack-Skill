# 下載文件檔
  - 打開[iClass](https://iclass.tku.edu.tw/)
  - 進入文件預覽畫面
  - F12 → Elements → Ctrl+F → pdf-viwer
  - 找到網址並複製下來
  - 到[URL解碼網站](https://www.convertstring.com/zh_TW/EncodeDecode/UrlDecode)
  - 把網址貼上去
# 下載影片檔
  - 打開[iClass](https://iclass.tku.edu.tw/)
  - 進入影片預覽畫面
  - F12 → Network → Media → Ctrl+R
  - 在下方視窗中找到影片檔
  - 按右鍵 → Open in new tab
